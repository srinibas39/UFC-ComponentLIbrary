Modal

The modal component provides a solid foundation for creating dialogs, popovers, lightboxes, or whatever else.

Basic Modal

Basic Modal that you can embed in your website by using class "dialog-container".

UFC DIALOG HEADER

UFC Dialog body text

    
//Basic Modal:

<div class="dialog-container">
<div class="dialog">
<h1>UFC DIALOG HEADER
<p>UFC Dialog body text

<div class="buttons">
<button class="action-btn">Action1
<button class="action-btn">Action2
</div>
</div>
</div>

List Modal

The following List Modal that you can embed in your website .

UFC DIALOG HEADER

  • Item-1
  • Item-2
  • Item-3

//Basic Modal:

<div class="dialog-container">
<div class="dialog2">
<h1>UFC DIALOG HEADER
<ul class="dialog-ul">
<li class="sel"><i class="fas fa-user-circle"></i>Item-1</li>
<li><i class="fas fa-user-circle"></i>Item-2</li>
<li><i class="fas fa-user-circle"></i>Item-3</li>
</ul>
</div>
</div>
