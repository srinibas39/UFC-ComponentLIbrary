Badge

The Badges are used to add additional information to the content. For example, some websites associate some number of notifications to the link. The notification number is seen when logged in to a particular website which tells the numbers of news or notifications to see by clicking it.

Text With a Badge

Text with a badge to add additional information to the text with class badge-icon.

Hello this is UFC

1

//Text With a Badge:

<div class="badge">
<h1 class="h1">Hello this is UFC</h1>
<div class="badge-icon">1</div>

Button toggles badge visibility

When you click on the button badge gets invisible and vice versa.

#

//Button toggles badge visibility:

<div class="badge">
<button> class="button">UFC </button>
<div class="button-badge">#</div>

//Javascript code to toggle

let button=document.querySelector(".button");
let icon=document.querySelector(".button-badge");
const toggleFn=()=>{
    if(icon.style.display!=="none"){
        icon.style.display="none";
        button.innerText="SHOW"
    }
    else{
        icon.style.display="flex";
        button.innerText="HIDE"
    }
}
button.addEventListener("click",toggleFn);
