Grid

The grid creates visual consistency between layouts while allowing flexibility across a wide variety of designs.

Grid Layout-1

This is a Grid layout-1 which you can use in your website.It is responsive by nature.

Header
Aside-1
Main
Aside-2
footer
    
//Grid Layout-1:

<div class="grid">
<div class="grid-header">Header</div>
<div class="grid-aside">Aside-1</div>
<div class="grid-main">Main</div>
<div class="grid-aside2">Aside-2</div>
<div class="grid-footer">footer</div>
