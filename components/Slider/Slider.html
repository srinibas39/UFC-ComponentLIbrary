Slider

Sliders allow users to make selections from a range of values.

Basic Slider

This is a basic Slider which you can use in your website.It is responsive by nature.It will allow you make selection from a range of values. 

    
//Basic Slider:

<div class="slider-container">
<input type="range" min="1" max="100" value="50" class="slider">
</div>

Sizes Slider

Different sizes of slider using classes like "sm","md","lg" for small ,medium ,large respectively. 


//Sizes Slider:

<div class="slider-container">
<input type="range" min="1" max="100" value="50" class="slider sm">
<input type="range" min="1" max="100" value="50" class="slider md">
<input type="range" min="1" max="100" value="50" class="slider lg">
</div>
