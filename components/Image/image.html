Image

The following are different kinds of responsive Images which you can embed in your webpage by using different classes.

Square Image

You can embed square images in your websites by using class "sqr-img". 


//Square Image:

<div class="sqr-img">  
<img src="../../images/conor.jpg" alt="loading...">  
<img src="../../images/Khabib.jpg" alt="loading...">
<img src="../../images/Gsp.jpeg" alt="loading...">  
</div>

Circular Image

You can embed circular images in your websites by using class "circular-img". 


//Circular Image:

<div class="circular-img">  
<img src="../../images/conor.jpg" alt="loading...">  
<img src="../../images/Khabib.jpg" alt="loading...">
<img src="../../images/Gsp.jpeg" alt="loading...">  
</div>

Sizes Image

You can embed images of different sizes in your websites by using class "sm","md","lg". 


//Sizes Image:

<div class="sizes-img">  
<img class="sm" src="../../images/conor.jpg" alt="loading...">  
<img class="md" src="../../images/Khabib.jpg" alt="loading...">
<img class="lg" src="../../images/Gsp.jpeg" alt="loading...">  
</div>

Big Image

You can embed Big image in your websites by using class "big-image".It can be used as banner. 


//Big Image:

<img class="big-image" src="../../images/UFC.png" alt="loading...">
