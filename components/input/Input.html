Input

The following are different kinds of responsive Input components which you can embed in your webpage by using different classes. You can use input component in searchbar,authentication,forms etc.

Search Input

You can embed search input in your websites by using class "search". 


//Search:

<input type="search" class="search" 
placeholder="&#xf689; Search" style="padding: 1rem; 
font-family:Montserrat ,sans-serif, 'Font Awesome 5 Free';">

Authentication Form

You can embed Authentication Form in your websites by using class "username" ,"password"

Wrong password.Try again. 

//Authentication Form:

<label class="label" for=".username">username *</label>
<input type="search" class="username" placeholder="Username">
<label class="label red" for=".username">password *</label>
<input type="search" class="password red red-border"
 placeholder="Password">
<small class="red">Wrong password.Try again.</small>

Radio Input

You can embed radio boxes in your website.It helps to select one item out of a group of items" 


//Radio:

<div class="checked">

<label class="label" for="checked">checked</label>
<input type="radio" name="checked" checked>
</div>
<div class="unchecked">
<label class="label" for="unchecked">unchecked</label>
<input type="radio" name="checked">
</div>
<div class="disabled">
<label class="label" for="disabled">disabled</label>
<input type="checkbox" name="disabled" disabled>
</div>

Checkbox Input

You can embed Checkbox in your website.It helps to select multiple item out of a group of items and you can also select and deselect same item mulitple times" 


//Radio:

<div class="checked">

    <label class="label" for="checked">checked</label>
    <input type="checkbox" name="checked" checked>
  
</div>
<div class="unchecked">
    <label class="label" for="unchecked">unchecked</label>
    <input type="checkbox" name="checked">
</div>
<div class="disabled">
    <label class="label" for="disabled">disabled</label>
    <input type="checkbox" name="disabled" disabled>
</div>
